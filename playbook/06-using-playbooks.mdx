Once you've created a playbook, you can deploy it to evaluate any deal.

## Two Ways to Use Playbooks

|                              | Description                                      |
| ---------------------------- | ------------------------------------------------ |
| **From Playbooks tab**       | Select a playbook, then choose a deal to evaluate |
| **From a Deal**              | Open a deal, then run a playbook on it           |

## Option 1: From the Playbooks Tab

<Steps>
  <Step title="Open Playbooks">
    Go to the left panel and click **Playbooks**.
  </Step>
  <Step title="Select your playbook">
    Click on the playbook you want to use.
  </Step>
  <Step title="Click Evaluate Opportunity">
    Click the **Evaluate Opportunity** button.
  </Step>
  <Step title="Select a deal">
    Click the **Select a Deal** button and choose the deal you want to evaluate.
  </Step>
</Steps>

<Info>
  You can add more playbooks to the same evaluation by clicking **Add a Playbook**.
</Info>

## Option 2: From a Deal

<Steps>
  <Step title="Open the deal">
    Go to the deal you want to evaluate.
  </Step>
  <Step title="Click Run Playbook">
    Click the **Run Playbook** button.
  </Step>
  <Step title="Select a playbook">
    A pop-up window appears. Choose the playbook you want to use.
  </Step>
</Steps>

## Evaluating with Multiple Playbooks

You can run multiple playbooks on the same deal. This is useful when:

- You want to see how a deal scores under different strategies
- You have different risk profiles (e.g., Value Add vs. Opportunistic)
- Different team members have different investment criteria

Just click **Add a Playbook** to include more playbooks in your evaluation.
