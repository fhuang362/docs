Once you purchase a model or skill from the Marketplace, it is available in your Library and ready to use across your workflows.

## Using Purchased Models

<Steps>
  <Step title="Open a Deal or Chat">
    Navigate to the Deal or Chat where you want to use the model.
  </Step>
  <Step title="Import from Library">
    Open the Library panel and select the purchased model to import it.
  </Step>
  <Step title="Work with the model">
    Use the model as a starting point — customize inputs, run scenarios, or adapt it to your specific deal.
  </Step>
</Steps>

<Tip>
  Save any modifications back to the Library if you want to reuse your customized version later.
</Tip>

## Using Purchased Skills

Skills extend what Apers can do in your conversations.

<Steps>
  <Step title="Open a Chat or Deal Chat">
    Start a new chat or open an existing one.
  </Step>
  <Step title="Activate the skill">
    Select the purchased skill from your available skills. Apers will use it to enhance its responses.
  </Step>
  <Step title="Ask questions or run analysis">
    Chat normally. The skill augments how Apers responds — providing specialized analysis, formatting, or domain knowledge.
  </Step>
</Steps>

## Quick Reference

| Action                        | Where                     |
| ----------------------------- | ------------------------- |
| Find purchased items          | Library                   |
| Import a model into a deal    | Library panel in any Deal |
| Use a skill in conversation   | Chat skill selector       |
| Share with your team          | Automatic via Library     |

<Warning>
  Skills apply to the chat where you activate them. If you want to use a skill in a different chat, activate it there as well.
</Warning>
