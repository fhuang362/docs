Workflows let you automate repetitive tasks. Define the steps once, then run them anytime with a single click.

## Where to Find Workflows

Click **Workflows** in the left side panel. From here you can:

- View your existing workflows
- Create new workflows
- Use templates built by the Apers team

## Two Ways to Create a Workflow

|                          | Description                                              |
| ------------------------ | -------------------------------------------------------- |
| **Create New Workflow**  | Build your own workflow from scratch                     |
| **Create from Template** | Start with a pre-built workflow from the Apers team      |

<Info>
  Templates are a great starting point. You can duplicate and customize them to fit your needs.
</Info>
