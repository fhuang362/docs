Files in your Library can be imported into any Chat or Deal.

## Importing Files

<Steps>
  <Step title="Open a Chat or Deal">
    Start a new Chat or open an existing Deal.
  </Step>
  <Step title="Find the Import button">
    Look for the **Import from Library** button on the bottom-right side of the Data Room.
  </Step>
  <Step title="Select your file">
    Choose the file you want to use.
  </Step>
  <Step title="Start working">
    The file loads into your Data Room, ready for Apers to use.
  </Step>
</Steps>

## What You Can Import

Anything stored in your Library:

- Financial models
- Word document templates
- Market assumptions
- Previous deal files
- Any other saved documents

## How Apers Uses Imported Files

Once imported, Apers can:

- Read and understand the file
- Use it as a starting template
- Reference its data or assumptions
- Modify it based on your instructions

<Info>
  Imported files work just like any other file in your Data Room. Apers treats them the same way.
</Info>

## Exporting Files to the Library

You can also save files from any Deal back to your Library for future reuse.

<Steps>
  <Step title="Open your Deal">
    Navigate to the Deal containing the file you want to save.
  </Step>
  <Step title="Find the file in the Data Room">
    Locate the file in the right-side Data Room panel.
  </Step>
  <Step title="Click the triple-dot icon">
    Click the **⋮** icon on the file.
  </Step>
  <Step title="Select Save to Library">
    Click **Save to Library**. The file is now available in your Library for use in other Deals.
  </Step>
</Steps>

<Tip>
  Export your best models, templates, and research files to the Library so you can reuse them across future deals.
</Tip>
