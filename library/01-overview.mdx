The Library is your firm-level knowledge management system. Store files here and reuse them across all your future deals.

## What You Can Store

- Market assumptions
- Financial models
- Word document templates
- Any other reusable files

## Accessing the Library

Click the **Library** tab on the left side panel to open your Library.

## Key Features

|                          | Description                                                  |
| ------------------------ | ------------------------------------------------------------ |
| **Upload files**         | Drag and drop files directly into the Library                |
| **Save from Chats**      | Push files from any Chat or Deal into the Library            |
| **Import to Deals**      | Pull files from the Library into any Chat or Deal            |
| **Firm-wide access**     | Everyone in your firm can access the same Library            |

<Info>
  The Library creates a feedback loop. As you build models and templates, save them so your firm can reuse them later.
</Info>
